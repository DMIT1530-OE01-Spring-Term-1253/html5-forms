## Introduction

Organize, build, and style a form for a painting company. The information they want to gather is below.

1.	Full Name
2.	Message
3.	Email Adress
4.	Phone Number
5.	Do you Agree to Terms/Be contacted (CASL compliance) – see what others have it as
6.	Which rooms to paint, the size, whether baseboards and ceilings need to be done as well
a.	kitchen
b.	bathroom(s) – max 2
c.	living room
d.	dining room
e.	bedroom(s) – max 3
f.	other – can type whatever want
7.	Preferred week to have work done
8.	Preferred method of contact
9.	Address

You must decide the types of inputs to use, and which fields are required before the form is submitted. Based on your knowledge of form best practices, identify those fields. 

A layout has not been provided. 

This form should look good on any size screen/resolution and must be fully accessible to screen readers. 

## The Build Methodology 
1.	Group and order the fields. Do not build the form in the order above. It is not user friendly.
2.	Choose the most appropriate input type of each field.
3.	Decide on a min and max widths for the form. Do not just stack as it’s a long form.
4.	Build the form HTML. Do not include a header or footer.
5.	Test, validate and repair any deficiencies.
6.	Submit your lab via the method noted in Brightspace.
